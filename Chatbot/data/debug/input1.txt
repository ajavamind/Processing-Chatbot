code a processing.org sketch in the java programming language that does the following:    the input is an List of strings from a file in the data folder of the sketch.    The sketch shows each string on the screen with size of 1024 width by 1024 height using text method.    Each line wraps around and advances from the top of the screen.    The strings can be arbitrary length, therefore put text on next line if it does not fit.    put a blank line between each string displayed.
